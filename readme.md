# Volumetric Flame Scene

<img width="347" height="301" alt="Знімок екрана 2026-07-02 о 12 38 07" src="https://github.com/user-attachments/assets/e80119e7-7eb4-45bb-b2d2-d961ce7037e0" />

An interactive 3D scene built with **Three.js**, rendering a volumetric 
flame effect with dynamic animation and smooth transitions between scenes.

## Features
- 🌫 Volumetric rendering with procedural noise (Perlin/Simplex)
- 🎨 Custom GLSL shaders for gradient coloring (blue/orange tones)
- 🔄 Scene-to-scene transition

## Stack
- Three.js
- WebGL / GLSL shaders
